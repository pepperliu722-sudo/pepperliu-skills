# pepperliu-skills

Reusable Codex skills for product demos, growth, and Humanizer.help workflows.

## Product Demo Video Storyteller

Create high-resolution product use-case videos that reproduce the real product UI one-to-one and demonstrate each feature's actual input, action, output, and evidence. It prevents generic UI, incorrect controls, unrelated result screens, unreadable output, and visual mismatch with the source website.

Download [product-demo-video-storyteller.zip](./product-demo-video-storyteller.zip) to install the first skill package.
